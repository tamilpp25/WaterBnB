# How to start
1. Clone Repository
```powershell
git clone https://github.com/MoonlightPS/MoonlightTS.git
```
2. Install Dependencies
```powershell
npm install
```

3. Start the server
```powershell
npm start
```
4. Use Postman to send requests to http://localhost:5000/
